## How to use
First import to your project by \#include
```c
#include '<path to CHeader.h file>'
```

### printfln() -- print line 

like a printf but add a new line at last. \
example code:
```c
printfln("Hello, world!");
printfln("Hello, world!");
printfln("Hello, world!");
```
output:
```
Hello, world!
Hello, world!
Hello, world!
```
